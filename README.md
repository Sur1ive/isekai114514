## 异世界野兽先辈

异世界野兽先辈是由大鹅自主研发的一款全新开放世界冒险游戏。 游戏发生在一个被称作「异世界」的幻想世界，在这里，被神选中的人将被授予「p眼」，导引池沼之力。玩家将扮演一位名为「野兽先辈」的神秘角色，在自由的旅行中邂逅性格各异、能力独特的同伴们，和他们一起击败远野，找回失散的亲人，同时逐步发掘「仲夏夜」的真相。
